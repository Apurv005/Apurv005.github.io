A Chromebook is a laptop or tablet running the Linux-based Chrome OS as its operating system. The devices are primarily used to perform a variety of tasks using the Google Chrome browser, with most applications and data residing in the cloud rather than on the machine itself. All Chromebooks released since late 2017 can also run Android apps. Some Chromebooks can run Linux apps. The first Chromebooks for sale, by Acer Inc. and Samsung, began shipping on June 15, 2011.In addition to laptop models, a desktop version, called a Chromebox.
